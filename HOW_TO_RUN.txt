PREREQUISITES
================================================================================
1. Python 3.11 or higher (for backend)
2. Node.js 16+ and npm (for frontend)
3. API Keys:
   - OpenAI API Key (get from https://platform.openai.com/api-keys)
   - Pinecone API Key (get from https://www.pinecone.io/)


SETUP & RUN INSTRUCTIONS
================================================================================

STEP 1: Configure Backend Environment
--------------------------------------
1. Navigate to backend folder:
   cd backend

2. Create .env file with your API keys:
   Create a file named ".env" in the backend/ folder with:

   OPENAI_API_KEY=sk-... (your OpenAI key)
   PINECONE_API_KEY=... (your Pinecone key)
   PINECONE_ENVIRONMENT=gcp-starter
   PINECONE_INDEX_NAME=acme-docs

3. Create Python virtual environment:
   python3 -m venv venv
   source venv/bin/activate  (On Windows: venv\Scripts\activate)

4. Install Python dependencies:
   pip install -r requirements.txt

5. Load documents into Pinecone (only needs to be done once):
   python load_documents.py

   Expected output:
   ✨ Loading complete! Total chunks stored: 3


STEP 2: Start Backend Server
-----------------------------
1. Make sure you're in the backend folder and venv is activated
2. Run the server:
   python main.py

   Expected output:
   INFO:     Uvicorn running on http://127.0.0.1:8000
   ✅ Successfully connected to Pinecone index
   
3. Test backend (in another terminal):
   curl http://localhost:8000/health

   Should show: {"status":"ok",...}


STEP 3: Setup Frontend
----------------------
1. Open a NEW terminal window
2. Navigate to frontend folder:
   cd frontend

3. Install npm dependencies:
   npm install

4. Start development server:
   npm run dev

   Expected output:
   ➜ Local: http://localhost:3001/


STEP 4: Use the Chatbot
-----------------------
1. Open your browser to: http://localhost:3001
2. You should see the chat interface
3. Type a question about Acme Tech Solutions, for example:
   - "When was Acme Tech Solutions founded?"
   - "What products does Acme Tech offer?"
   - "What is the remote work policy?"

4. Click "Send" or press Enter
5. The chatbot will search documents and respond with an answer


TESTING THE APPLICATION
================================================================================

Test Backend Directly:
----------------------
1. In a terminal, make sure backend is running
2. Test health endpoint:
   curl http://localhost:8000/health

3. Test chat endpoint:
   curl -X POST http://localhost:8000/api/chat \
     -H "Content-Type: application/json" \
     -d '{"question": "When was Acme Tech Solutions founded?"}'

Expected response:
{
  "success": true,
  "answer": "2015",
  "sources": [...]
}


TROUBLESHOOTING
================================================================================

Backend Won't Start:
- Ensure Python 3.11+ is installed: python3 --version
- Check virtual environment is activated
- Verify .env file exists in backend/ with API keys
- Check firewall allows port 8000

Frontend Won't Connect:
- Ensure backend is running on port 8000
- Check browser console (F12) for errors
- Verify Vite is running on port 3001
- Clear browser cache if needed

No Results from Queries:
- Confirm documents are loaded: check backend logs
- Verify Pinecone credentials in .env are correct
- Check that index exists in Pinecone dashboard
- Try reloading documents: python load_documents.py

API Key Issues:
- Ensure OPENAI_API_KEY starts with "sk-"
- Ensure PINECONE_API_KEY is not empty
- Test OpenAI: curl -H "Authorization: Bearer YOUR_KEY" https://api.openai.com/v1/models
- Test Pinecone in Python: python -c "import pinecone; pinecone.init(...)"

STOPPING THE APPLICATIONS
================================================================================

To stop backend:
- Press Ctrl+C in the terminal running "python main.py"

To stop frontend:
- Press Ctrl+C in the terminal running "npm run dev"

To deactivate Python virtual environment:
- Type: deactivate
